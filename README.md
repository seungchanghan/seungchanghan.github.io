# Seungchang Han Portfolio

Personal research portfolio built with React and Vite.

## Development

```bash
npm ci
npm start
```

## Verification

```bash
npm test
npm run check-format
npm audit
```

## Deployment

Pushes to `main` build the site and deploy `dist/` to the `gh-pages` branch.
